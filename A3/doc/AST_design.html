The indentation indicates which classes extends which other ones.

compiler488.ast.SourceLoc/h4>

Contains information about a portion of the source code. We added this class.

compiler488.ast.SourceLocPrettyPrinter/h4>

Contains methods to print info about where an error occurred. We added this class.

compiler488.ast.Printable

Interface, indicates that the AST can be an argument to a put statement.

compiler488.ast.Readable

Interface, indicates that the AST can be an argument to a get statement.

compiler488.ast.AST

Root superclass for all syntax tree nodes. We added a parent-children interface allowing node traversal, that is, a member called parent and a method called getChildren. Added source location tracking.

compiler488.ast.IdentNode

extends AST. Represents an identifier instead of just using a String. Added this class.

compiler488.ast.IntConstNode

extends AST. Represents an integer constant value instead of using a Int. Added this class.

compiler488.ast.ArrayBound

extends AST. Represents the bounds of an array instead of using two ints. Added this class.

compiler488.ast.ASTList<E>

extends AST. A list of nodes. Delegated rather than subclassed because of a language restriction.

compiler488.ast.expn.Expn

compiler488.ast.expn.ConditionalExpn

compiler488.ast.expn.FunctionCallExpn

compiler488.ast.expn.ConstExpn

compiler488.ast.expn.IntConstExpn

compiler488.ast.expn.NewlineConstExpn

compiler488.ast.expn.BoolConstExpn

extends ConstExpn. The reserved words `true' and `false'. Added source location tracking.

compiler488.ast.expn.TextConstExpn

extends ConstExpn, implements Printable. A double-quoted string. Added constructor, source location.

compiler488.ast.expn.UnaryExpn

compiler488.ast.expn.NotExpn

compiler488.ast.expn.UnaryMinusExpn

extends UnaryExpn. Added constructor, source location.

compiler488.ast.expn.BinaryExpn

compiler488.ast.expn.ArithExpn

compiler488.ast.expn.EqualsExpn

extends BinaryExpn. Added constructor, operator constant strings, source location tracking.

compiler488.ast.expn.CompareExpn

compiler488.ast.expn.BoolExpn

compiler488.ast.expn.SubsExpn

compiler488.ast.stmt.Program

compiler488.ast.stmt.ReturnStmt

compiler488.ast.stmt.ResultStmt

compiler488.ast.stmt.IfStmt
